# OMH Skills — Legacy (v1)

These are the v1 skill implementations that work **without the OMH plugin**.
They are verbose but fully functional — state management and evidence gathering
are spelled out in prose rather than delegated to plugin tools.

## When to use these

- You want to use OMH skills without installing the plugin
- You are on a Hermes installation where plugins are not available

## v2 skills (plugin required)

The skills in `skills/` (parent directory) are v2 — they require the OMH plugin
at `plugin/` in this repo. Install it by symlinking or copying to
`~/.hermes/plugins/omh/`.

See the root README for installation instructions.
